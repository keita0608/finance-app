# 設定・データ管理

e-shiwakeでは「設定」と「データ管理」を別ページに分けています。

| ページ                | 場所                        | 内容                                   |
| --------------------- | --------------------------- | -------------------------------------- |
| 設定（`/settings`）   | サイドバー「設定」→「設定」 | 事業者情報・証憑保存先・ストレージ容量 |
| データ管理（`/data`） | サイドバー「データ管理」    | バックアップ・リストア / エクスポート  |

## 設定ページ（`/settings`）

### 事業者情報

青色申告決算書に使用する事業者情報（氏名・屋号・住所等）を登録します。

### 証憑保存設定（年度別）

証憑PDFの保存先を**年度ごと**に設定できます。同じ端末内でも年度によって保存先を使い分けられます。

| 保存先           | 対応ブラウザ | 特徴                                                            |
| ---------------- | ------------ | --------------------------------------------------------------- |
| ローカルフォルダ | Chrome, Edge | ファイルとして直接保存。ブラウザ容量を消費しない                |
| ブラウザ内       | すべて       | IndexedDBに保存。Safari/Firefox対応。エクスポートで取り出し可能 |

> **INFO**: デフォルトの保存先はブラウザの対応状況で自動判定されます。Chrome/Edgeでは「ローカルフォルダ」、Safari/Firefoxでは「ブラウザ内」が既定値です。

#### 年度別の保存先切替

設定ページの「年度別の証憑保存先」で、各年度の保存先をローカルフォルダ⇔ブラウザ内に切り替えられます。切替時には該当年度の証憑PDFが自動的に移行（マイグレーション）されます。

- ローカル→ブラウザ: ファイルをIndexedDBに取り込み、ローカルファイルを削除
- ブラウザ→ローカル: IndexedDBからファイルとして書き出し、Blobを削除

移行は確認ダイアログの後に実行され、進捗バーで状況を確認できます。

> **TIP**: リストア時に選択した保存先も年度別の設定として自動保存されます。

#### フォルダ設定の注意

ローカル保存モードでは、保存先フォルダの指定が必要です。
フォルダが未設定の場合、証憑PDFの参照・保存ができません。

以下の場合にフォルダ設定が失われることがあります:

- 別の端末やブラウザでデータをインポートした場合
- ブラウザのデータが初期化された場合

この場合、同じフォルダを再選択すれば証憑へのアクセスが復活します。

> **WARNING**: フォルダ設定をクリアすると、ローカルフォルダ内の証憑PDFへのリンクが切れます。証憑ファイル自体はフォルダに残りますが、仕訳との紐付けが失われます。

### データの保存場所

e-shiwakeのデータは、ブラウザ内のIndexedDBに保存されます。
サーバーへのデータ送信は行いません。

| データ           | 保存先                                          |
| ---------------- | ----------------------------------------------- |
| 仕訳メタデータ   | IndexedDB（共通）                               |
| 証憑PDF（実体）  | IndexedDB または ローカルフォルダ（設定に依存） |
| 勘定科目・設定等 | IndexedDB（共通）                               |

> **WARNING**: ブラウザのデータ消去やシークレットモードでは、データが失われる可能性があります。定期的なバックアップをお勧めします。

### ストレージ使用量

ブラウザ内（IndexedDB）に証憑を保存している場合、ストレージ容量を消費します。設定ページでは現在の使用量と推定上限が確認できます。

この情報は主に、Safari/Firefoxなど容量上限が厳しい環境で、ブラウザ内保存を使用している場合の容量管理に役立ちます。

| ブラウザ      | 容量目安                |
| ------------- | ----------------------- |
| Chrome / Edge | ディスク空き容量の約80% |
| Safari        | 約1GB                   |
| Firefox       | ディスク空き容量の約50% |

#### 証憑ダウンロード（Blob Purge）

ブラウザ内保存モードで容量が逼迫した場合に、証憑PDFを個別にダウンロードしてIndexedDB上のBlobを削除する機能です。Safari等の容量制限が厳しい環境での応急措置として用意されています。

通常の証憑管理には以下の方法が適しています:

- 個別のPDF確認: 仕訳のPDFリンクからダウンロード
- 一括保存: バックアップ（ZIP形式で全データ+証憑を保存）
- 年度末: アーカイブ（帳簿レポート+証憑+検索HTMLを一括保存）
- 容量回復: ローカルフォルダ保存への切替（年度別に設定可能）

> **TIP**: Chrome/Edgeでは年度別の保存先をローカルフォルダに切り替えることで、ブラウザ容量を消費せずに証憑を保存できます。確定申告後はアーカイブを作成してから年度データを削除する方法もあります。

## データ管理ページ（`/data`）

データ管理は以下の2層で構成されています。

| 層                     | 形式       | 対象                   | 用途                              |
| ---------------------- | ---------- | ---------------------- | --------------------------------- |
| バックアップ・リストア | ZIP        | 全データ＋証憑PDF      | 端末移行・事故対策                |
| CSVインポート          | CSV        | 仕訳データ             | Excelで作成した仕訳の一括登録     |
| エクスポート           | JSON / CSV | 仕訳データ（証憑なし） | Excel確認・他ソフト連携・外部保存 |

これに加えて、年度末の長期保存用にアーカイブ機能（`/archive`）があります。

| 機能       | 形式 | 内容                               |
| ---------- | ---- | ---------------------------------- |
| アーカイブ | ZIP  | 仕訳＋証憑＋帳簿レポート＋検索HTML |

### CSVインポート

ExcelなどのCSVファイルから仕訳を一括登録できます。

**CSVフォーマット（列順）**

| 列名         | 必須 | 説明                                          |
| ------------ | ---- | --------------------------------------------- |
| 日付         | ○    | YYYY-MM-DD または YYYY/MM/DD 形式             |
| 摘要         |      | 仕訳の説明                                    |
| 取引先       |      | 取引先名                                      |
| 借方科目     | ○    | 科目名またはコード番号                        |
| 借方金額     | ○    | 税込金額（整数）                              |
| 貸方科目     | ○    | 科目名またはコード番号                        |
| 貸方金額     | ○    | 税込金額（整数）                              |
| 消費税区分   |      | 省略時は科目のデフォルト区分が適用            |

**複合仕訳の入力方法**

2行目以降の「日付」列を空欄にすることで、同一の仕訳に複数の借方・貸方明細を追加できます。

```
日付,摘要,取引先,借方科目,借方金額,貸方科目,貸方金額,消費税区分
2024-04-01,立替精算,○○商事,旅費交通費,5500,,, 課仕10%
,,,消耗品費,3300,,,課仕10%
,,,,, 未払金,8800,
```

**消費税区分の指定値**

| 入力値    | 意味          |
| --------- | ------------- |
| 課売10%   | 課税売上10%   |
| 課売8%    | 課税売上8%    |
| 課仕10%   | 課税仕入10%   |
| 課仕8%    | 課税仕入8%    |
| 非課税    | 非課税        |
| 不課税    | 不課税        |
| 対象外    | 対象外        |

> **TIP**: データ管理ページで「テンプレートCSV」と「勘定科目コード一覧」をダウンロードできます。Excelで編集後「CSV UTF-8」形式で保存してからインポートしてください。

> **INFO**: 添付ファイル（証憑PDF）はCSVインポートの対象外です。インポート後に仕訳帳から個別に追加してください。

> **TIP**: 通常のデータ保全にはバックアップを、確定申告後の年度締めにはアーカイブを使い分けてください。詳細は各ヘルプページをご覧ください。
